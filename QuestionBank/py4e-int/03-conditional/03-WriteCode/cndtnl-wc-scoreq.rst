    Write a program to prompt for a score between 0.0 and 1.0. If the score is out of range,
    print an error message. If the score is between 0.0 and 1.0, print a grade using the
    following table:

    .. code-block::

        Score   Grade
       >= 0.9     A
       >= 0.8     B
       >= 0.7     C
       >= 0.6     D
        < 0.6     F


    .. code-block::

       Enter score: 0.95
       A
