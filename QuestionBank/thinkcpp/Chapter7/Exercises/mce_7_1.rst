    Which variable below is declared as a ``string`` type?
    
    .. code-block:: cpp
    
     int main() {
       int x = 0;
       double y = 4.5;
       string word = "hello";
       char c = 'c';
       bool isPrime = 1;
     }
    
    - ``x``
    
      - ``x`` is an ``int``.
    
    - ``y``
    
      - ``y`` is a ``double``.
    
    - ``word``
    
      + ``word`` is a ``string``.
    
    - ``c``
    
      - ``c`` is a ``char``.
    
    - ``isPrime``
    
      - ``isPrime`` is a ``bool``.
