    Use a two-level dictionary to store information on members of your extended family.

    The top-level dictionary should have the last name of the person as its key. Its value should be another dictionary. The lower-level dictionary should have the person's name should be the key and their age should be the value.

    Write code to:

    1. Create the dictionaries with your family member's information.

    2. Find the average age of people with each last name.

    3. Find the name of the oldest person with each last name.
