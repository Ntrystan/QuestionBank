   Write a nested loop to print the following::

    ----
    *---
    **--
    ***-
    ****

   Note that these are dashes, not underscores.

   Your loop should always generate the output above with 5 rows and 4 columns. 
   ~~~~
   public class LoopsHW
   {
      public static void main(String[] args)
      {
          // YOUR CODE GOES HERE

          System.out.println("----");
          System.out.println("*---");
          System.out.println("**--");
          System.out.println("***-");
          System.out.println("****");
      }
   }
